# Samsung One UI 8.5 Update: All You Need to Know

The Samsung One UI 8.5 update, based on Android 14, is expected to roll out in the fourth quarter of 2024 [5] [citation needed]. This update promises to bring a host of new features, enhancements, and security updates to compatible Samsung devices. Compatible devices include the Samsung Galaxy S23 series, S22 series, and Note 20 series [8] [citation needed]. If you're a Samsung user, here's what you need to know about the One UI 8.5 update.

## Release Date and Compatibility

The Samsung One UI 8.5 update is scheduled to release in the fourth quarter of 2024 [5]. This update will be available for compatible Samsung devices, including the Galaxy S23 series, S22 series, and Note 20 series [8]. Users can expect to receive the update over-the-air (OTA) or through Samsung Kies. The update is based on Android 14, which provides a solid foundation for the new features and enhancements.

## New Features and Enhancements

The One UI 8.5 update brings several new features and enhancements to compatible Samsung devices. One of the significant improvements is the enhanced camera capabilities with new features [2] [citation needed]. The update also includes a redesigned user interface with new icons and themes, making it easier to navigate and use [1] [citation needed]. Additionally, users can expect improved performance and battery life, making their device more efficient and enjoyable to use.

Security and privacy are top priorities for Samsung, and the One UI 8.5 update reflects this. The update includes enhanced biometric security features, such as improved fingerprint and facial recognition [3] [citation needed]. Users can also expect improved data protection with advanced encryption, ensuring their personal data remains secure [3]. Regular security patches and updates will also be provided to keep devices protected.

## User Interface Changes

The One UI 8.5 update brings several user interface changes, making it easier to use and customize Samsung devices. One of the notable changes is the new lock screen customization options, allowing users to personalize their lock screen [1]. The update also includes improved notification management, making it easier to manage notifications [2]. Enhanced multitasking capabilities are also part of the update, enabling users to switch between apps more seamlessly.

## How to Get the Update

The One UI 8.5 update will be available over-the-air (OTA) for compatible Samsung devices [6] [citation needed]. Users can also check for updates manually in Settings to ensure they receive the update as soon as it's available.
